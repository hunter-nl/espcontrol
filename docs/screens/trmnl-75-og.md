# TRMNL 7.5 OG

The **TRMNL 7.5 OG** is a 7.5-inch, 800 x 480 monochrome e-paper display powered by an ESP32-S3.

For the full walkthrough, see the [Install guide](/getting-started/install).

::: tip E-paper refresh
TRMNL uses an e-paper display, so EspControl batches screen updates. Use the **Refresh Display** control if you want to force a redraw while testing.
:::
